# F-1 學生稅務問題清單 (模擬網頁流程)

這份文件將引導您完成一系列問題，以確定您的報稅需求。每個頁面模擬了在一個網站或應用程式上您可能會遇到的步驟。

---

## Page 1: 個人與居住資訊 (Personal & Residency Information)

**頁面目標：** 收集您的基本身份、居住地和在美停留時間，以確定您的稅務身份和適用的州稅法規。

**問題 1:**
*   **問題:** 您首次以 F-1 學生身份進入美國是哪一年？
*   **互動方式:** `日期選擇器 (Date Picker)` 或 `下拉式年份列表 (Dropdown Year List)`。

**問題 2:**
*   **問題:** 在上一個納稅年度，您總共在美國境內停留了多少天？
*   **互動方式:** `數字輸入框 (Numeric Input Field)`。

**問題 3:**
*   **問題:** 您的國籍（護照簽發國）是？
*   **互動方式:** `可搜尋的國家下拉式列表 (Searchable Country Dropdown)`。此資訊將用於自動查找可能的稅收協定 (Tax Treaty)。

**問題 4:**
*   **問題:** 您目前居住在哪個州？
*   **互動方式:** `美國各州下拉式列表 (US States Dropdown)`。

---

## Page 2: 稅務身份識別 (Taxpayer Identification)

**頁面目標：** 確認您是否擁有報稅所需的身份識別號碼。

**問題 1:**
*   **問題:** 您是否擁有社會安全碼 (SSN) 或個人納稅識別號碼 (ITIN)？
*   **互動方式:** `是/否 單選按鈕 (Yes/No Radio Buttons)`。

**問題 2 (條件性顯示):**
*   **觸發條件:** 如果上一題回答「是」。
*   **問題:** 請輸入您的 SSN 或 ITIN。
*   **互動方式:** `文字輸入框 (Text Input)`，輸入內容應被遮蔽以保護隱私。

---

## Page 3: 收入詳情 (Income Details)

**頁面目標：** 了解您在美國的收入來源和類型，以確定需要填寫哪些收入相關的稅務表格。

**問題 1:**
*   **問題:** 在上一個納稅年度，您是否有任何來自美國的收入？
*   **互動方式:** `是/否 單選按鈕 (Yes/No Radio Buttons)`。

**問題 2 (條件性顯示):**
*   **觸發條件:** 如果上一題回答「是」。
*   **問題:** 您的收入來源包括哪些？(可複選)
*   **互動方式:** `複選框 (Checkboxes)`，選項包括：
    *   [ ] 校內工作 (On-campus Employment)
    *   [ ] 獎學金/助學金 (Scholarship / Fellowship)
    *   [ ] 銀行存款利息 (Bank Interest)
    *   [ ] 其他 (Other)

**問題 3 (條件性顯示):**
*   **觸發條件:** 如果上一題回答「是」。
*   **問題:** 根據您的收入來源，您收到了以下哪些表格？(可複選)
*   **互動方式:** `複選框 (Checkboxes)`，選項包括：
    *   [ ] **Form W-2** (來自雇主)
    *   [ ] **Form 1042-S** (來自學校或獎學金發放機構)
    *   [ ] **Form 1099-INT** (來自銀行)
    *   [ ] **Form 1099-MISC / 1099-NEC** (其他)

---

## Page 4: 文件生成與總結 (Document Generation & Summary)

**頁面目標：** 根據您提供的所有資訊，自動生成預覽的稅務表格，並提供下載選項。

**此頁面沒有問題，而是顯示結果。**

**顯示內容:**
*   **摘要區塊:** 簡要總結用戶輸入的關鍵資訊。
*   **待辦事項列表:**
    *   [x] 確定稅務身份為：**Nonresident Alien**
    *   [x] 將為您生成 **Form 8843**
    *   [x] (如有收入) 將為您生成 **Form 1040-NR**
    *   [x] (如適用) 將為您生成 **[州名] 州稅表格**

**互動方式:**
*   `下載按鈕 (Download Button)`：點擊後，用戶可以下載一個包含所有已填寫（草稿）表格的 PDF 檔案。
*   `郵寄說明 (Mailing Instructions)`：清晰地列出聯邦和州稅務部門的郵寄地址以及截止日期。
